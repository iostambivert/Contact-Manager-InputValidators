i cant upload the installer file because its too big for github, but one can be made by just being in the same folder as the main.py, installing pyinstaller packaage and running this command
```
pyinstaller --onefile main.py
```
this will create a dist/ folder which will have a file called "main". if we copy this file anywhere and run it, it will function as a normal application without any other files needed
